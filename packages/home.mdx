# 组件库文档指南

> 工欲善其事必先利其器，文档工具是所有开发多人合作的必备利器，它解决了成员的沟通、标准化等问题，大大提高开发人员的工作效率

目前组件库分为几类

1. 工具类组件库 `@nbit-utils`
2. React 组件库 `@nbit-react`
3. TradingView 组件库 `@nbit-tradingview`
4. 通用图表组件库 `@nbit-chart`
5. yapi to typescript 组件库 `@nbit-ytt`

新增的组件要先写在项目里，成熟后抽离到 toolbox 仓库

[查看所有 npm 内部包](https://npm.nbttfc365.com/)
